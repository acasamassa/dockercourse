# Docker - Compose

To check options:

`docker-compose`

Create and start containers:

`docker-compose up -d`

Check on browser:

`http://localhost:8088`

Check containers:

`docker ps`

Check networks:

`docker network ls`

Check volumes:

`docker volume ls`

Stop and remove containers, networks:

`docker-compose down`

Delete volumes:

`docker volume rm 11-03-compose_db 11-03-compose_wp`
